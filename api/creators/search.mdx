# Search Creators

Search for creators across multiple platforms with advanced filtering, faceted search, and full-text capabilities.

## Endpoint

```
POST /api/creators/search
```

## Request

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Must be `application/json`
</ParamField>

### Body Parameters

<ParamField body="query" type="string">
  Search query for creator name, bio, or tags
</ParamField>

<ParamField body="filters" type="object">
  Optional filters to narrow down results
  
  <Expandable title="Filter Options">
    <ParamField body="platform" type="string">
      Filter by platform: `youtube`, `twitter`, `instagram`, `tiktok`, `linkedin`
    </ParamField>
    
    <ParamField body="category" type="string">
      Filter by category: `tech`, `lifestyle`, `gaming`, `beauty`, `fitness`, etc.
    </ParamField>
    
    <ParamField body="tags" type="string[]">
      Array of tags to filter by
    </ParamField>
    
    <ParamField body="minFollowers" type="number">
      Minimum follower count across all platforms
    </ParamField>
    
    <ParamField body="maxFollowers" type="number">
      Maximum follower count across all platforms
    </ParamField>
    
    <ParamField body="minEngagement" type="number">
      Minimum composite engagement score (0-100)
    </ParamField>
    
    <ParamField body="maxEngagement" type="number">
      Maximum composite engagement score (0-100)
    </ParamField>
    
    <ParamField body="verified" type="boolean">
      Filter by verification status
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="sort" type="object">
  Sorting configuration
  
  <Expandable title="Sort Options">
    <ParamField body="by" type="string">
      Sort field: `relevance`, `followers`, `engagement`, `recent`
    </ParamField>
    
    <ParamField body="order" type="string">
      Sort order: `asc` or `desc`
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="pagination" type="object">
  Pagination settings
  
  <Expandable title="Pagination Options">
    <ParamField body="page" type="number" default="1">
      Page number (1-indexed)
    </ParamField>
    
    <ParamField body="limit" type="number" default="20">
      Results per page (1-100)
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="searchType" type="string" default="combined">
  Search algorithm: `fulltext`, `fuzzy`, or `combined`
</ParamField>

### Multi-Platform Search

```typescript
// Search creators with presence on multiple platforms
const response = await fetch('https://your-domain.com/api/creators/search', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${apiKey}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    query: 'lifestyle',
    filters: {
      tags: ['fashion', 'beauty', 'wellness'],
      minFollowers: 50000
    },
    sort: {
      by: 'engagement',
      order: 'desc'
    }
  })
});

const { data } = await response.json();

// Filter creators with multiple platform presence
const multiPlatformCreators = data.creators.filter(creator => 
  Object.keys(creator.platforms).length >= 3
);
```
