# Welcome to TikTok Miner

TikTok Miner is a comprehensive creator discovery and analytics platform that helps you find and evaluate content creators based on real engagement metrics and trending topics.

## What is TikTok Miner?

TikTok Miner automates the process of discovering high-performing content creators by:

1. **Searching** TikTok using keywords and hashtags
2. **Scraping** creator profiles and their recent content
3. **Aggregating** 30-day engagement metrics (views, likes, comments, shares)
4. **Analyzing** performance data in a real-time dashboard
5. **Exporting** results for further analysis

## How It Works

<Steps>
  <Step title="Search for Creators">
    Enter keywords, hashtags, or trending topics to search TikTok
  </Step>
  <Step title="Automatic Scraping">
    The platform scrapes the top creators from search results using Apify
  </Step>
  <Step title="Metrics Aggregation">
    Collects and aggregates engagement data from the last 30 days of posts
  </Step>
  <Step title="View & Export">
    Browse results in the dashboard, filter by metrics, and export to CSV
  </Step>
</Steps>

## Tech Stack

TikTok Miner is built with modern, production-ready technologies:

- **Frontend**: Next.js 14, React, TailwindCSS, shadcn/ui
- **Backend**: Node.js, Prisma ORM, PostgreSQL
- **Queue System**: BullMQ with Redis
- **Scraping**: Apify actors for TikTok, Instagram, Twitter, YouTube
- **Infrastructure**: Docker, Docker Compose
